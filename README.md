# xcx-cameraselector
CameraSelector extension for [Xcratch](https://xcratch.github.io/).
This is an extension that allows select video devices in Scratch.

## How to Use in Xcratch
This extension can be used with other extension in [Xcratch](https://xcratch.github.io/).

- Open [Xcratch Editor](https://xcratch.github.io/editor)
- Click 'Add Extension' button
- Select 'Extension Loader' extension
- Type the [module URL](https://tfabworks.github.io/xcx-cameraselector/dist/cameraselector.mjs) in the input field

```
https://tfabworks.github.io/xcx-cameraselector/dist/cameraselector.mjs
```

## for Development

Follow these basic steps to use this project:

```bash
# Setup for Scratch
# This command sets up the Scratch-related repositories
# and installs necessary dependencies.
npm run setup-scratch

# Build Process
# This command executes the build of the project.
# It generates files for production use.
npm run build

# Starting the Application
# This command launches the development server and
# makes the project viewable in a browser.
npm run start
```
