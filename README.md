# xcx-cameraselector
Scratch3から使用するカメラデバイスを切り替えられるようにする拡張です。

## Usage

```
# 下準備 (install scratch-gui and scratch-vm)
./scripts/setup-scratch.sh

# register to local scratch
npm run register

# Start local dev server ( http://127.0.0.1:8601 )
./scripts/dev-server.sh

# Build mjs
./scripts/build.sh

```
